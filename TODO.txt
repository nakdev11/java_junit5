品種とサイズを与えて、いちごを作成し、その文字列表現を取得しましょう。

文字列表現の例: あまおう: S とちおとめ: M もういっこ: L

なお、品種は、 あまおう とちおとめ もういっこ の3種類、いちごのサイズは S M L LL の4種類です。

- 品種とサイズを与えて、いちごを作成する
  - [x] 品種を文字列で与えて、あまおう　であればいちごクラスに保存
  - [x] 品種を文字列で与えて、とちおとめ　であればいちごクラスに保存
  - [x] 品種を文字列で与えて、もういっこであればいちごクラスに保存
  - [x] 品種を文字列で与えて、あまおう とちおとめ もういっこ　ではない場合　例外を返す

  - [x] サイズを文字列で与えてS M L LLであればいちごクラスに保存
  	- [x]サイズを文字列で与えてSであればいちごクラスに保存
  	- [x]サイズを文字列で与えてMであればいちごクラスに保存
  	- [x]サイズを文字列で与えてLであればいちごクラスに保存
  	- [x]サイズを文字列で与えてLLであればいちごクラスに保存
  - [x] サイズを文字列で与えてS M L LLでない場合　例外を返す

- [ ] いちごの文字列表現を取得する
  - [x] 品種が登録されている場合、登録された品種を {品種}: {サイズ}　の文字列で取得できる
  - [ ] いちごの品種、サイズが登録されていない場合
    - [x] サイズのみ登録されている場合、 不明: {サイズ}　を文字列で取得できる
    - [x] 品種のみ登録されている場合、 {品種}: 不明 　を文字列で取得できる
    - [x] 品種、サイズが登録されていない場合、 不明: 不明 　を文字列で取得できる



    いちごは重さによってサイズが決定されます。
    重くなるにしたがってサイズが大きくなります。

    サイズではなく、重さを直接与えて、いちごを作成できるようにしましょう。

    ただし、いちごの重さは 1g以上とし 0g以下は対象としなくてよいものとします。

    なお、いちごのサイズは、重さによって決まります。

    重さ と サイズ の関係性は以下のとおりです。

    25g以上 → LL
    20g以上 → L
    10g以上 → M
    1g以上 → S
    例えば、 重さが 8g の あまおうの場合は、 あまおう: S という文字列表現になります。

    * 重さの単位
      * グラム(g)固定？

   * [ ] 重さを数字で与えて、いちごクラスに保存できる
     * [ ] 重さを数字で与えて、重さが25g以上の場合サイズがLLで保存される
     * [ ] 重さを数字で与えて、重さが20g以上25g未満の場合サイズがLで保存される
     * [ ] 重さを数字で与えて、重さが10g以上20g未満の場合サイズがMで保存される
     * [ ] 重さを数字で与えて、重さが1g以上10g未満の場合サイズがSで保存される
